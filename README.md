# Glavos Linux

Source [glavos.github.io](https://glavos.github.io)

Fork [Just the Docs](https://pmarsceill.github.io/just-the-docs/).

## Installation

[Ruby#Setup](https://wiki.archlinux.org/index.php/Ruby#Setup).

```bash
sudo pacman -S ruby

gem install bundler

bundle install
```

```bash
bundle exec jekyll serve -l
```
